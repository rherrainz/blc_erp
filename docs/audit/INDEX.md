# Auditoría - Índice

Bienvenido al submódulo de documentación de auditoría. Aquí encontrarás las políticas, la implementación técnica y guías de uso para consultar y mantener los registros de auditoría.

Documentos incluidos:

- `POLICY.md`         : Política de auditoría y requisitos.
- `IMPLEMENTATION.md` : Detalles técnicos de la implementación (modelos, middleware, signals).
- `HOWTO.md`          : Guías rápidas para administradores y desarrolladores (consultas, export, restauración).
- `FAQ.md`            : Preguntas frecuentes relacionadas con auditoría.

Empieza por `POLICY.md` para entender el alcance y las obligaciones.
